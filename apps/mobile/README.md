# @blowcortex/mobile

Application mobile BlowCortex — Expo SDK 52 + Expo Router 4.

## Démarrage

```bash
cd apps/mobile
pnpm dev          # ouvre Metro et le QR code Expo Go
pnpm ios          # iOS simulator (macOS uniquement)
pnpm android      # Android emulator
```

## Statut Sprint 1

Squelette uniquement : un écran d'accueil minimal pour valider que le
scaffolding Expo + le monorepo pnpm fonctionnent ensemble.

L'auth Clerk, le feed (lecture seule), les briefings et les notifications push
arrivent au Sprint 4 (PRD §9.4).
